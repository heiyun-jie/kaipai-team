# 00-22 设计说明

## 1. 设计原则

- 保持后台账号页现有信息结构，不扩大改动面
- 先修操作列容器与换行约束，不调整业务动作语义
- 延续 `/dashboard/index` 已建立的后台视觉节奏

## 2. 设计策略

### 2.1 操作列容器

- 为操作按钮增加统一包裹容器，改为多行弹性布局
- 重置 link button 在表格操作区的默认外边距，避免换行后出现错位
- 控制按钮行高、间距和单项不折字，保证标签边界稳定

### 2.2 fixed 列稳定性

- 保持操作列固定在右侧，避免运营滚动查看时丢失关键动作
- 通过列宽约束和容器换行共同收口 fixed 列内容宽度

## 3. 影响文件

- `kaipai-admin/src/views/system/AdminUsersView.vue`
