# 05-06 当前主线残余清理（Mainline Residual Cleanup）

> 状态：进行中 | 优先级：高 | 依赖：05-02, 05-05

## 1. 目标

在名片分享 + 会员分层重构之后，继续清理当前主线中的残余文案、假数据和不一致表达，避免当前分支继续带着旧方案痕迹运行。

## 2. 本轮范围

- `pages/mine/index`
- `pages/actor-profile/edit`
- 当前主线相关工具与状态映射

## 3. 需求

### 3.1 身份文案收敛

- **R1** 当前主线页面不得继续使用没有明确业务含义的旧英文身份文案，例如 `PRO ACTOR / ACTIVE ACTOR / ACTOR / TEAM`。
- **R2** 演员身份、档案状态、会员状态必须拆开表达，不得混用一枚文案同时表达多个维度。

### 3.2 假数据清理

- **R3** “我的”页不得展示无真实来源的 fake 统计数据，例如固定浏览量 `1.2k`。
- **R4** 当前展示统计必须来自现有可用数据，或改为稳定的结构化指标。

### 3.3 主线一致性

- **R5** “我的”页、档案编辑页与名片页的身份表达需要保持当前主线一致。
- **R6** 清理过程中不得重新引入信用积分、等级组件或旧入口。

## 4. 验收标准

- [ ] 当前主线页面已移除 `PRO ACTOR / ACTIVE ACTOR / ACTOR / TEAM` 这类残余身份文案
- [ ] “我的”页不再展示无来源 fake 数据
- [ ] 清理后当前主线页面仍可通过类型检查
- [ ] 当前主线页面仍无信用积分 / 排行榜残余引用
