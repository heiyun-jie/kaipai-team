# 05-01 演员明信片（Actor Card）

> 注：本 Spec 为早期名片方案，当前主线已由 `05-05-card-share-membership` 接管。

> 状态：待实现 | 优先级：高 | 依赖：03-04, 04-04

## 1. 功能概述

演员可生成个人介绍明信片，分享到微信好友/朋友圈。他人点击可进入小程序查看演员完整档案。

## 2. 需求清单

### 2.1 明信片展示页 `pages/actor-card/index`

- **R1** 展示演员核心信息：头像（大尺寸）、姓名、性别、城市、擅长类型标签
- **R2** 数据栏展示：演员等级（LV.X + 名称）、积分 + 徽章（预留，05-03）、完成拍摄次数
- **R3** 代表照片区：从照片墙中展示 3 张代表照
- **R4** 自我介绍摘要：截取前 100 字
- **R5** 底部展示小程序码 + "扫码查看完整档案"提示

### 2.2 分享能力

- **R6** 转发好友：`onShareAppMessage` 携带 `actorId` 参数，分享卡片展示演员头像+姓名
- **R7** 分享朋友圈：`onShareTimeline` 支持
- **R8** 生成海报：Canvas 绘制完整明信片图片，支持保存到相册
- **R9** 海报包含小程序码，扫码可直接跳转演员详情

### 2.3 落地页改造 `pages/actor-profile/detail`

- **R10** 去除身份限制，改为公开可访问（不再限制 `UserRole.Crew`）
- **R11** 增强展示内容：完整照片墙、视频、作品经历、演员等级标签、积分徽章
- **R12** 非登录用户可浏览，投递/联系需登录

### 2.4 入口

- **R13** 「我的」页面（`pages/mine/index`）新增"分享我的名片"菜单项
- **R14** 演员资料编辑页（`pages/actor-profile/edit`）底部新增"预览名片"按钮

## 3. 页面类型

- 明信片页：B 类（普通顶部页）
- 落地页：A 类（深色 Hero 页）
