# 00-35 设计说明

## 1. 设计原则

- 不改后端口径，只增强前端可见性
- 把“当前筛查影响范围”放在统计卡之前，先说明口径，再展示数据
- 让空态也遵循当前筛查条件，而不是继续使用固定描述

## 2. 设计策略

### 2.1 范围摘要

在筛查面板下方新增两块范围摘要：

- 统计范围
- 最近事项范围

展示规则：

- 时间窗口未设置时，说明“未额外收窄时间窗口 / 按系统默认口径”
- 时间窗口已设置时，展示当前 `dateFrom/dateTo`
- `bizLine` 未设置时，说明最近事项展示全部业务线
- `bizLine` 已设置时，展示当前业务线名称
- 统计范围卡固定提示：`bizLine` 不影响统计卡

### 2.2 最近事项空态

最近事项为空时，不再只显示固定“当前没有最近事项”，而是按当前筛查条件给出更具体的提示：

- 无筛查：当前没有最近事项
- 仅时间窗口：当前时间窗口下没有最近事项
- 有业务线：当前筛查条件下，该业务线没有最近事项
