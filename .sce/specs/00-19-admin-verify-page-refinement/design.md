# 00-19 设计说明

## 1. 设计原则

- 不只修单个输入框，而是按“审核工作台”的页面模型重新组织这一页
- 保持现有后台暖色基调，但提高表单对比度和局部信息密度
- 新增内容优先服务运营判断，不堆叠技术说明

## 2. 设计策略

### 2.1 概览区

- 在页头下方补一层轻量概览，说明当前页面处于待审核或历史回看模式
- 用卡片提示当前筛选状态、列表规模和处理动作

### 2.2 筛选区

- 改为当前页专用筛选网格，标签上置
- 输入框、下拉框和日期控件统一使用更强的表面色、边界和阴影
- 重置 / 查询动作与筛选字段分层摆放

### 2.3 主表格与空态

- 表格增加当前列表说明和空态提示块
- 空态使用运营语义文案，减少“空白表格”的观感

### 2.4 详情区

- 详情抽屉补充状态摘要与说明区
- 信息块继续使用共享 detail block，但加上当前页的审核语义

- `kaipai-admin/src/styles/index.scss`
